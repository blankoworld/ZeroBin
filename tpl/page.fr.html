ZeroBin est un outil de partage de texte en ligne dont le code source est ouvert. Le serveur où réside les données n'a aucune connaissance du contenu. Les informations sont chiffrées/déchiffrées dans le navigateur en utilisant le standard de chiffrement avancé (AES) à 256 bits. Plus d'informations sur la page du projet.
▶ Note: Ceci est un service de test: Les données peuvent être supprimées à n'importe quel moment. Des chats mourrons si vous abusez de ce service.
